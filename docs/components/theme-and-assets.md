# Тема и ресурсы рендера

## `src/render/theme.js`

Модуль экспортирует единый объект `theme`. Рендерер читает его напрямую; runtime-переопределение через переменные окружения не предусмотрено.

### Цвета

- `background: #16233f` — фон bubble и хвоста;
- `userColors` — восемь цветов имени и fallback-аватара;
- текст сообщения всегда белый.

Цвет автора выбирается детерминированно по `abs(Number(user.id)) % userColors.length`.

### Layout

| Параметр | Значение | Назначение |
| --- | ---: | --- |
| `leftPadding` | 18 | Левое поле canvas и позиция аватара |
| `tailWidth` | 28 | Ширина хвоста bubble |
| `tailOverlap` | 7 | Перекрытие хвоста и bubble |
| `topPadding` | 24 | Верхнее поле |
| `sameSenderGap` | 6 | Интервал внутри группы автора |
| `senderGap` | 22 | Интервал между авторами |
| `stickerFooter` | 82 | Прозрачный низ итогового WebP |

### Текст

Основной шрифт имеет базовый размер 31 px, line-height 37 px. При оптимизации компоновки перебираются размеры от 34 до 29 px. Контент ограничен 40…430 px, кандидат ширины меняется шагом 14 px.

`wrapChars`, `groupedWidthRatio` и `groupedWidthCap` присутствуют в теме, но текущий `quote.js` их не использует. Их изменение не повлияет на результат.

### Медиа и аватары

Обычное медиа ограничено 560×560. Круглое video note имеет 320×320 и внутренний отступ bubble 30 px.

Тема задаёт набор размеров аватара для плотного текста, абзаца, группы, обычного/короткого текста, медиа и круглого медиа. Фактический размер выбирает рендерер по длине и числу строк группы.

## `assets/fonts`

В комплект входят:

- `NotoSans-Regular.ttf` — weight 400;
- `NotoSans-SemiBold.ttf` — weight 600;
- `NotoSans-Bold.ttf` — weight 800;
- `NotoSans-Italic.ttf` — italic 400;
- `NotoSans-SemiBoldItalic.ttf` — italic 600.

Файлы синхронно читаются во время импорта `quote.js` по пути `join(process.cwd(), 'assets/fonts/...')`. Поэтому:

- все пять файлов обязательны для запуска;
- рабочая директория должна быть корнем репозитория;
- ошибка чтения возникает при старте, до обработки первого Update.

Bold italic entity фактически запрашивает weight 800 italic, хотя отдельного такого font face нет; итоговый выбор/синтез выполняет Satori.

## Изменение внешнего вида

После изменения чисел темы достаточно перезапустить приложение: build step отсутствует. При изменении размеров стоит проверить короткий и длинный текст, нескольких авторов, caption с фото, стикер и video note. Автоматических snapshot-тестов в проекте нет.
