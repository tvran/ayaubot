# AyauBot — документация проекта

AyauBot — Telegram-бот для групповых чатов. Он превращает сообщения в статические WebP-стикеры, скачивает Instagram Reels и TikTok-видео, ведёт статистику слов, запускает игру с кодовым словом, напоминает о днях рождения и ежедневно выбирает случайного участника чата.

Проект написан на Node.js в формате ES modules и может работать как Vercel Function или как обычный HTTP-сервер.

## Быстрый старт

Требуется Node.js с глобальными `fetch`, `FormData` и `Blob` (на практике Node.js 18+), Telegram-бот и доступный извне HTTPS URL. Для загрузки Reels/TikTok также нужны исполняемые `yt-dlp` и `ffmpeg`.

```bash
npm install
cp .env.example .env
npm start
```

Локальный сервер принимает обновления по `POST /telegram/webhook`, а Vercel-функция — по `POST /api/telegram`. Перед регистрацией webhook заполните как минимум `BOT_TOKEN`, `WEBHOOK_SECRET` и `APP_URL`, затем выполните:

```bash
npm run set-webhook
```

Для локального сервера дополнительно задайте `WEBHOOK_PATH=/telegram/webhook`. Полная таблица настроек приведена в [руководстве по настройке](setup-and-operations.md).

## Команды бота

| Команда | Назначение |
| --- | --- |
| `/q` | Создать цитату из сообщения, на которое отвечает команда |
| `/q 2` … `/q 10` | Включить в цитату до 10 последовательных сообщений |
| `/qs` | Сохранить созданный ботом статический стикер в групповой стикерпак |
| `/qd` | Удалить стикер, на который отвечает команда, из его набора |
| `/top`, `/topwords` | Показать пять самых частых слов за последние 14 дней |
| `/pidor` | Получить детерминированный на текущий день случайный выбор участника |
| `/pidor_list` | Показать последние 15 ежедневных выборов |
| `/pidor_reset`, `/reset_pidor` | Сбросить сегодняшний выбор |
| `/codeword_start` | Запустить или продолжить трёхдневную игру |
| `/codeword` | Показать статус игры |
| `/codeword_hint` | Показать длину слова и самого частого пользователя слова |
| `/codeword_stats` | Показать рейтинг победителей игры |
| `/codeword_stop` | Завершить активную игру |
| `/birthday ДД.ММ.ГГГГ` | Записать или обновить свою дату рождения в чате |
| `/birthdays` | Показать календарь дней рождения чата |
| `/birthday_remove` | Удалить свою дату из календаря чата |
| `/help`, `/start` | Показать встроенную справку |

Сообщение с поддерживаемой ссылкой на Instagram Reels или TikTok обрабатывается без отдельной команды: бот скачивает ролик и отвечает на исходное сообщение видеофайлом.

Команды допускают суффикс имени бота, например `/q@my_bot`. Аналитические функции доступны только при настроенном PostgreSQL; создание многострочных цитат требует Redis.
